## Giới thiệu đề tài

Đề tài **"Image-to-Instructions for Water Hyacinth Weaving"** (Chuyển đổi hình ảnh sang hướng dẫn đan lục bình) là một dự án nghiên cứu áp dụng tư duy tính toán để giải quyết bài toán tự động hóa quy trình tạo hướng dẫn đan lát thủ công.
